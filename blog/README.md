Ruby Grape Skeleton
===========

This is a skeleton project for Grape, a REST framework for Ruby.

## Initialise

Use Bundler to namage ruby gems.

To install the gems:

```shell
bundle install
```

## Test

To run the test using Rspec:

```shell
rspec
```

## Run

To run the app using rackup:

```shell
rackup config.ru
```
